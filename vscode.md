# shortcuts
- `ctrl` + `shift` + `/`: open new terminal
- `cmd` + `j`: toggles terminal
- `cmd` + `1`: focuses editor
- `cmd` + `shift` + `[`: focus previous terminal
- `cmd` + `shift` + `]`: focus next terminal
- `cmd` + `shift` + `q`: terminate focused terminal
- `ctrl` + `shift` + `l`: go to line
